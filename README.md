<h2 align="center">⚙️ How I Work</h2>

<p align="center">
<b>Design</b> — I work the product end to end in Figma: flows, UX, interface, the whole experience.<br/>
<b>Direct the build</b> — I hand those designs to AI build tools (Figma Make, Claude Code, Lovable) and direct them to production-ready front-end. Not coding by hand is a method, not a gap.<br/>
<b>Ship</b> — I deploy the working product to a live URL. One person, concept to launch.
</p>
